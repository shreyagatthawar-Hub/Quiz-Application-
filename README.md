# Quiz-Application

## Description
A responsive and interactive Quiz Application built using HTML, CSS, and JavaScript. It allows users to select quiz categories and customize the number of questions, with features like randomized questions, a countdown timer, instant answer feedback, and final score tracking. The project demonstrates strong concepts of DOM manipulation, event handling, and dynamic UI updates.


##  Features 
### 1. Category-Based Quiz System
Users can choose from multiple categories such as **Programming, General Knowledge, Mathematics, and Entertainment**.
Each category contains a dedicated set of questions, making the quiz more organized and topic-focused.

### 2. Custom Question Selection
Users can select how many questions they want to attempt:
* 5 questions
* 10 questions
* 15 questions
* 20 questions
This allows flexibility depending on time and difficulty preference.

### 3. Randomized Questions (No Repetition)
* Questions are selected randomly from the chosen category
* Already attempted questions are tracked
* Ensures **no duplicate questions** appear in a single quiz session

### 4. Countdown Timer for Each Question
* Each question has a **15-second timer**
* Timer updates every second in real-time
* When time runs out:
  * The correct answer is automatically highlighted
  * User input is disabled
* Adds a **challenge and urgency** to the quiz experience

### 5. Instant Answer Feedback
* As soon as the user selects an option:
  * Correct answer → Highlighted in **green**
  * Incorrect answer → Highlighted in **red**
* A visual icon is displayed:
  * ✔️ for correct
  * ❌ for incorrect
* Improves learning through immediate feedback

### 6. Answer Locking Mechanism
* Once an answer is selected:
  * All options become unclickable
* Prevents multiple selections or cheating
* Ensures fair evaluation

### 7. Real-Time Progress Tracking
* Displays current progress:
  ex.2 of 10 Questions
* Helps users keep track of their quiz journey

### 8. Score Calculation System
* Tracks number of correct answers
* Final score is calculated automatically
* Displayed at the end of quiz:
   ex.You answered 7 out of 10 correctly

### 9. Result Screen with Summary
* Shows:
  * Total correct answers
  * Total questions
* Provides a clear performance overview
* Encourages users to improve

### 10. Retry Quiz Functionality
* “Try Again” button resets:
  * Score
  * Question history
  * UI state
* Allows users to retake quiz without refreshing the page

## How to Run
* Select a Category
* Choose Number of Questions
* Click Start Quiz
* Answer questions within 15 seconds
* View final results
* Click Try Again to restart
